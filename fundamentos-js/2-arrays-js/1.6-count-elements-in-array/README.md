#### Javascript Arrays
Esta série de desafios testa seu conhecimento sobre as matrizes Javascript
#### Questão:
Escreva uma função que tenha uma matriz (a) como argumento. Retorne o número de elementos em a.

#### Casos de Teste:

- myFunction([1, 2, 2, 4])
    - Esperado: 4


- myFunction([9, 9, 9])
    - Esperado: 3


- myFunction([4, 3, 2, 1, 0])
    - Esperado: 5
