#### Javascript Arrays
Esta série de desafios testa seu conhecimento sobre as matrizes Javascript
#### Questão:
Escreva uma função que receba um array de strings como argumento. Retorna a string mais longa.


#### Casos de Teste:
- myFunction(['help', 'me'])
  - Esperado: 'help'
- myFunction(['I', 'need', 'candy'])
  - Esperado: 'candy'
